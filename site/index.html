Get it running

One-time setup, then never again

You'll need Claude Code. Paste two lines, then the only thing you ever type is a sentence.

  1. Add the marketplace
    /plugin marketplace add mikeparcewski/wicked-interactive
  2. Install the plugin
    /plugin install wicked-interactive
  3. From then on, just say it

    “start wicked-interactive”

    Your browser pops open and you're off. The first time, it shows you exactly which helper tools it's setting up — nothing sneaky.

Rather install the helpers yourself? Set WI_NO_AUTOINSTALL=1 and it'll just tell you what to run.
