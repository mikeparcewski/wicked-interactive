# Craft references

Curated document-craft guidance, absorbed from wicked-prezzie when it merged into
wicked-interactive (ADR-0020). The supervising agent draws on these when generating a first
draft (assist Step 5) or making a whole-document change (Step 4) — the *method*, not a
workflow engine.

- **[outline-method.md](outline-method.md)** — structure first: the Pyramid Principle, how to
  turn raw material into a lead + supporting points before any HTML exists.
- **[story-arc.md](story-arc.md)** — narrative shapes for decks, one-pagers, and launch pages.
- **[html-craft.md](html-craft.md)** — the HTML/CSS constraints that keep a generated document
  clickable (data-wid friendly), themeable, and export-clean; plus image sourcing.

These are guidance, not gates. The hard rules live in the skill (preserve every `data-wid`,
ground claims in wicked-brain). Use judgment.
